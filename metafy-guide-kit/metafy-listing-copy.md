# Metafy Listing Copy

These are templates. Replace bracketed text with your deck details.

## Positioning recommendation

If you want maximum publicity:

- Start free if you are building your name from scratch.
- Start at $5-$10 if you already have results, a following, or VOD support.

## Title formulas

- `[Deck Name]: The Complete Guide`
- `How I Built [Deck Name] to Beat [Meta Target]`
- `[Deck Name] Deck Guide for [Set/Format]`
- `The Rogue [Deck Name] Guide`
- `[Deck Name] - Mulligans, Matchups, and Game Plan`

## Best title rule

Lead with the deck name first. Then add the promise.

Strong example:

`[Deck Name] - Full Matchup and Mulligan Guide`

Weak example:

`A Different Way to Think About Riftbound`

## Short description template

"This guide covers my [Deck Name] list for [format/set], including the deck’s core game plan, card choices, mulligan decisions, matchup approach, and the key mistakes players should avoid. I built this version to [solve a problem / target a meta], and I’ll keep refining the guide as testing continues."

## Punchier description template

"I built this [Deck Name] list to punish [meta trend] while keeping strong play into [important matchup]. Inside, I break down the decklist, explain the unusual card choices, show how I mulligan, and outline how I approach the matchups that matter most."

## What-you-get bullets

Use 4-6 bullets max.

- Full decklist breakdown
- Mulligan priorities
- Matchup guide
- Flex slots and tech options
- Common mistakes and pilot tips
- Updates as the list evolves

## Credibility paragraph template

"This guide comes from direct testing and refinement, not theory alone. I created this list after [testing volume/result], and the goal is to give players a practical, usable plan for picking up the deck quickly and improving with it fast."

## CTA lines

- "If you try the list, I’d love to hear how it performs for you."
- "I’ll keep updating this guide as the deck evolves."
- "This is the best place to follow how I’m refining the archetype."

## Pricing angle copy

### If free

"I’m releasing this guide for free because I want more people testing the deck, talking about it, and helping refine the archetype."

### If low-ticket

"I priced this guide low on purpose so it’s easy to jump in, try the deck, and support future updates."

### If premium

"This guide is meant to be a full reference for serious players who want matchup planning, deeper sequencing, and a guide that stays updated as the format changes."
